# 令虎｜年代MAP（学年版）

- GitHub Pages でそのまま公開可能な静的サイト
- **学年ベース（世代）**の一覧。敬称略
- 生まれ年版は `data-born.js` を用意して別タブに増設可能

## 公開手順（GitHub Pages）
1. このフォルダを新規リポジトリにアップロード（例: `reitora-age-map`）
2. Settings → Pages → Source を「Deploy from a branch」で `main / root` に設定
3. 数十秒後に公開URLが発行される

## データの更新
- `data.js` の配列を編集するだけ
- 例: `"84": ["林", "ホワイトニングカフェ佐藤"]` のように追記
- 59〜99、00 をキーとして扱っています（文字列）

## 仕様
- 検索: 入力に部分一致
- ソート: 昇順/降順切替
- 配色: ダーク/ライト切替（ローカル保存）
